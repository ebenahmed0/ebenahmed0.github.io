Some computer security tips:

  • Use a discreet password manager. Bitwarden is good if you need cloud sync, KeePassXC is better if you don't.
  • Make sure file name extensions are turned on in the file manager settings, otherwise it can be difficult to tell the difference between a normal file/folder and a well-disguised executable program.
  • Avoid installing software using a web browser. This is the best chance someone has to get their unwanted software on your PC. Scoop is a nice alternative if you're too cool for the Windows Store.
